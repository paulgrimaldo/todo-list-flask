# Todo List with Python 3.7!

This app was made with **Python** ,**Flask**, and **Google Cloud Firestore**.
List of requirements: 

 - flask-wtf  
- flask-testing  
- blinker  
- firebase-admin  
- flask-login

Install requirements. 

    pip install -r requirements.txt

Run main.py 

    python main.py
